# how to
* create `.env` file
* start project `docker compose up -d`

# .env file vars
* `PG_VERSION=15.5`
* `PG_USER=user`
* `PG_PASSWORD=password`
* `PG_DB=teamcity`

* `TEAMCITY_VERSION=2023.11.4`
* `TEAMCITY_UI_PORT=80`

* `DB_CPU=4`
* `DB_RAM=4G`
* `DB_SHM=2gb`

* `SERVER_CPU=4`
* `SERVER_RAM=4G`
* `SERVER_SHM=2gb`

* `AGENT_CPU=1`
* `AGENT_RAM=1G`
* `AGENT_SHM=0.5gb`
