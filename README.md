# how to
* create `.env` file
* start project `docker compose up -d`

# .env file vars
* `PG_VERSION=15.5`
* `PG_USER=user`
* `PG_PASSWORD=password`
* `PG_DB=teamcity`

* `TEAMCITY_VERSION=2023.11.1`
* `TEAMCITY_UI_PORT=80`

* `DB_CPU=2`
* `DB_RAM=4G`
* `SERVER_CPU=4`
* `SERVER_RAM=8G`
* `AGENT_CPU=1`
* `AGENT_RAM=2G`
